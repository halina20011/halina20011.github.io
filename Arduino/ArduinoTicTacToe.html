Arduino Tic Tac Toe

In this tutorial you can learn how to upload and run Tic Tac Toe game on Arduino Uno with 3.5" TFT Lcd touch screen.

Parts or tools you will need

Arduino Uno

3.5" TFT Lcd touch screen shield

Libraries

Download and put these libraries to Arduino libraries folder:

TFTLCD Library: https://github.com/adafruit/TFTLCD-Library
GFX Library: https://github.com/adafruit/Adafruit-GFX-Library
TouchScreen: https://github.com/adafruit/Adafruit_TouchScreen

Fixing screen size

Because TFT lcd library by default is set to wrong screen size you will need to change it first.

After downloading the library, go in it Arduino libraries folder and open file Adafruit_TFT/Adafruit_TFTLCD.cpp in our favourite text editor.

In file find lines that contains screen size and comment it. Alternatively you can just change the values.

                            //#define TFTWIDTH 240 
                            //#define TFTHEIGHT 320

Add these lines.

                            #define TFTWIDTH  320 
                            #define TFTHEIGHT  480

Save it end exit the text editor.

Circuit diagram

Just mount it on top of an Arduino board.

Code

Because code is quite big it is available on Github.

Download code with git. Alternatively you can download code with browser, but after that you will need to unzip it.

git clone https://github.com/halina20011/Arduino-TicTacToe.git

Navigate to TicTacToe subfolder and open it in Arduino IDE:

cd Arduino-TicTacToe/TicTacToe/
arduino TicTacToe.ino

In Arduino IDE select correct board and port and the press "upload button".
